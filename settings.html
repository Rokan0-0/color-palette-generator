ℹ️ About

Color Palette Generator Pro

Version 2.0.0

A professional-grade color palette generator designed for designers and developers. Create, customize, and export beautiful color schemes with advanced features.

✨ Key Features

  • 🎲 6 Color Harmony Modes
  • 📷 Image Color Extraction
  • 👁️ Color Blindness Simulation
  • ♿ WCAG Accessibility Checker
  • 🌈 Advanced Gradient Generator
  • 🎨 HSL & RGB Color Picker
  • 💾 Unlimited Cloud-Free Storage
  • 🔗 Share Palettes via URL
  • 📥 6 Export Formats
  • 🌓 Dark/Light Mode
  • ⌨️ Keyboard Shortcuts
  • 📱 Fully Responsive
